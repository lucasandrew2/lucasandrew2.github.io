Overview

This course taught me how to build full-stack web applications from the ground up and then attack them like a hacker would. Programming languages/frameworks I learned in this class were HTML5, CSS, JavaScript, PHP MySQL. Vulnerability classes I learned were cross-site scripting, SQL injection, session hijacking, and CSRF. This course also taught me secure programming principles so that my code can be functional and defensible.
